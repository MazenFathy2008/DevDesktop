Welcome to DevDesktop

The porpuse of this project

The idea for this project goes back to when I was working on another project. I remembered my past projects. Therefore, I wanted to create a good UI platform that contains my projects, but GitHub already does this, so I thought about what would make this platform unique, and I found it. It will be a full developer desktop containing a favorite IDE, favorite browser, and favorite OS. Also, it will contain a favorite music folder. In addition, the developer can make their desktop public so it can be shown to other developers. Also, they could share their desktop with their friends only, but this project needs a lot of effort. Therefore, I decided to build a small static demo first to collect feedback and enhance the idea before building the full project. So, after visiting the website, please send me your feedback in the feedback section.

This demo contains

  • Preferred Code Editor
  • Preferred Browser
  • Preferred Operating System
  • Projects
  • Preferred Music Folder
